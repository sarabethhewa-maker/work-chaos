# 🏃 Work Chaos

Put your coworkers in chaotic animated environments. Upload face photos, pick a scene, watch them run around and trip over each other.

## Features
- 6 hand-crafted environments: Beach, Forest, Mountains, Racetrack, City Rooftop, Garden
- Upload real face photos — they get clipped onto animated running bodies
- Physics-based chaos: characters collide, trip, get back up with stars 😵
- Up to 10 characters at once
- Live cast panel showing everyone's current state

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000), add your coworkers, pick a vibe 🌊🌲⛰️
